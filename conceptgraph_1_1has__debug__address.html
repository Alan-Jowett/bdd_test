Detailed Description

Optional: handle exposes a debug address.

Some handle types provide a debug_address() accessor that returns either a const void* or std::uintptr_t. This is only used for diagnostics and for producing stable human-readable IDs; it is optional for algorithms. Use if constexpr (has_debug_address<H>) to detect availability.
