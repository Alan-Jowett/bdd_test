Detailed Description

Generic template for generating node tables from BDD structures.

This header provides a generic template-based approach to generating node tables from BDD structures in multiple output formats. It uses concepts to define the interface requirements for iterators and supports various output formats including plain text tables and Markdown.

The iterator represents the current node and provides methods to access node properties such as variable names, child relationships, and terminal values.
