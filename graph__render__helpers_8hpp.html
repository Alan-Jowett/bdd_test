Detailed Description

Small helpers for rendering nodes and edges in graph output formats.

This header provides lightweight, header-only utilities used by the Mermaid and DOT generators to produce deterministic textual lines for nodes, edges and CSS class assignments. The helpers ensure consistent escaping of labels and stable ordering of class-to-node mappings which is important for the test-suite's byte-for-byte comparisons.
