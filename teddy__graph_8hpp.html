Detailed Description

TeDDy BDD graph generation using generic template systems.

Provides iterator interface and graph generation functions for visualizing Binary Decision Diagrams using the generic dot_graph_generator.hpp and mermaid_graph_generator.hpp templates. These functions produce identical output to the BDD-specific implementations but use the flexible template system.
