Detailed Description

CUDD BDD graph generation using generic template systems.

Provides iterator interface and graph generation functions for visualizing Binary Decision Diagrams using the generic dot_graph_generator.hpp and mermaid_graph_generator.hpp templates. These functions produce high-quality output for CUDD BDDs using the flexible template system.
