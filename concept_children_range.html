Detailed Description

Concept requiring a children range.

Requires that T exposes a get_children() member that returns a C++20 range whose range_value_t is convertible to T (i.e. the range yields elements that are iterator/node-like objects of the same type).

This accepts owned containers (for example std::vector<T> returned by value), borrowed views (std::span<T>, std::ranges::subrange, or other borrowed ranges) and range adaptors. If an implementation returns a view into internal storage, it is the implementer's responsibility to ensure the referred storage remains valid for the duration of use.
