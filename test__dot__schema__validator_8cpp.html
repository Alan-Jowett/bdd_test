Detailed Description

DOT schema validator used to validate generated .dot outputs.

This test provides a small but robust DOT syntax validator (parser) that verifies typical Graphviz DOT output produced by the project's generators. The validator checks for:

  • valid header (digraph/graph)
  • balanced braces
  • statements ending with semicolons
  • well-formed attribute lists ([k=v,...])
  • matched quoted strings
