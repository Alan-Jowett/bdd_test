Detailed Description

External DAG view concept.

Types that model external_dag_view expose a nested handle type and provide the following operations:

  • g.children(h) returns a children range yielding elements convertible to edge-like objects whose target() yields a handle.
  • optionally g.roots() returns an input-range of root handles.

This abstraction decouples algorithms from concrete graph storage implementations and enables generic traversal utilities.
