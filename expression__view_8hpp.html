Detailed Description

Adapter exposing the expression AST as a graph::external_dag_view.

This adapter provides a thin view over the variant-based AST defined in expression_types.hpp. It models graph::external_dag_view by providing nested handle, edge, and the children() / roots() methods used by generic graph algorithms in graph.hpp.
