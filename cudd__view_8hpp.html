Detailed Description

Adapter exposing a CUDD DdNode graph as a graph::external_dag_view.

This adapter wraps raw CUDD pointers (DdNode*) and presents a small view type that models graph::external_dag_view used by the generic graph algorithms. The adapter preserves CUDD complement-edge semantics by returning child handles that include the complement bit when appropriate.
