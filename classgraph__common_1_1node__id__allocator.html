Detailed Description

Allocates deterministic textual IDs for node pointers.

Example: with prefix "N" and start_index 1 the produced IDs are "N1", "N2", ...

This class is intentionally small and non-thread-safe. It is expected to be created per-document or per-generation run and not shared across threads.
